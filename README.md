

# TripPlanner

This project was generated using [Nx](https://nx.dev).

Technologies used:
- NX for monorepo
- ExpressJS (Custom) for backend
- Angular 14 for frontend
- Tailwind for styling
- Mapbox Gl for Geolocation rendering
